Dark Reign 1.8.2 Public Beta

This patch is a work in progress with the aim of fixing bugs and expanding the modability of the 1997 PC game Dark Reign. DarkUpdate.exe, placed in an empty directory, will guide you through the install process. Note that the process will request a Dark Reign 1.4 or higher install to reference for collecting base game files and that an internet connection is required during the update process. The GOG version of this install works best but other versions may be attempted. If you have any issues please let us know and be sure to provide the numerous .log files generated during your test run.
